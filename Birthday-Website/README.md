# Birthday Website Project

This project transforms the original single-page birthday experience into a modular, polished website structure.

## Structure

- index.html: main entry point
- style.css: all visual styling
- script.js: global interactions and bootstrapping
- data/: personalized content and lists
- components/: scene-specific modules
- assets/: images, music, videos, icons, and fonts

## Personalization

Edit the values in data/config.js to change:

- Girl name
- Nickname
- Anniversary
- My name
- Birthday
- Favorite color
- Favorite song path
- Relationship duration
- Photo paths
- Video paths
- Voice note paths
- Secret password

## Notes

- Replace placeholder files in assets with your own media when available.
- Open index.html in a browser to preview the experience.
